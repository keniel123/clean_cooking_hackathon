# GridCook AI Clean Cooking Dashboard

Static GitHub Pages dashboard for the Kenya-first clean cooking hackathon concept.

Live site target:

https://keniel123.github.io/clean_cooking_hackathon/

The dashboard presents:

- Kenya clean-cooking problem framing
- Oloika mini-grid and e-cooker evidence
- cooker profiles
- health-weighted smoke-exposure proxy
- AI scheduler concept
- daytime cooking rewards
- hackathon execution plan

This is a proof-of-concept. The health section estimates reduced smoke-exposure sessions as a proxy and does not claim medical diagnosis or measured disease reduction.
